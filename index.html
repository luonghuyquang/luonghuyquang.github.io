Course Objectives

The objective is to create user and mobile friendly, static website using web standards like HTML5 and CSS.
First we will study the basics and then in project groups, we will design and create a website.

Evaluation Criteria

The assessment is mainly based on the website project.
You can find more detailed assessment criteria from the website project assignment.

The website must be published on a web server. In HAMK we have a web server called shell.hamk.fi that you can use.
